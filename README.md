# React-Music-Player
React Music Player/react音乐播放器

## Usage

```bash
# 下载代码
git clone https://github.com/junhey/React-Music-Player.git
# 进入项目目录并安装依赖
cd React-Music-Player && npm install
# 运行预览
npm start
# 开发模式
npm run dev
# 编译发布
npm run bulid
```
